AskYQ

Static API

These endpoints are static JSON files served by GitHub Pages.

Base URL

https://abidlabs.github.io/askyq

Endpoints

  • API index
    /api/index.json
  • All fatwas (list with summaries)
    /data/fatwas.json
  • Single fatwa (full transcript)
    /api/fatwas/{id}.json

Example requests

curl https://abidlabs.github.io/askyq/data/fatwas.json
curl https://abidlabs.github.io/askyq/api/fatwas/insurance-in-islam.json
